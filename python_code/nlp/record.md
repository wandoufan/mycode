NLP目前的主要困难:
1.数据稀疏性
中文的字词组合非常多
2.形式和语义之间的多对多关系：
歧义性：一个词有多个意思
长距离依赖：窗口位置很远的词都会影响当前词的意思
3.理解语言通常需要背景知识和推理能力
