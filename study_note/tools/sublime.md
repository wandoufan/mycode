# sublime编辑器


## 常用快捷键

* ctrl+？ 可以给选中部分全部加#注释

* ctrl+shift+R  可以给选中代码全部改为指定格式

* ctrl+}或{  可以让选中部分代码整体左右移动缩进

* ctrl+shift+p  可以调出命令框，输入packag可以找出安装新库在内的所有包管理工具

## 常用sublime插件

* SideBarEnhancements  提供文件目录方面更详细的功能

* Python PEP8 Autoformat  提供PEP8代码格式规范
